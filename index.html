Application Scenarios

1. Multi-Site Enterprise Surveillance

A customer with over ten remote sites uses GV-RNVRL810-P at each location. Each NVR connects through GV-VPN to the central monitoring system at headquarters. Operators monitor all sites using GV-Edge Recording Manager on Windows PCs, while supervisors access footage on-the-go via the GV-Eye mobile app.

Before adopting GV-VPN, their IT department had to perform extensive network configurations — router setup, port forwarding, and security rules. With GV-VPN, all sites are securely interconnected with no manual configuration required, reducing maintenance time and improving response efficiency.

2. Mobile Surveillance Trailer

A customer in the mobile surveillance trailer business faced a major networking challenge.: their 4G/5G network only provided virtual IP addresses under the carrier’s APN, meaning the trailers could not be accessed directly from the Internet — even with DDNS services.

By installing a GV-Cloud Bridge inside each trailer, they securely linked all cameras to the GV-VPN mesh network using the trailers’ existing 4G/5G mobile connection. At their headquarters, operators used GV-Control Center through GV-VPN to monitor live feeds and coordinate emergency responses in real time.
